<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* rustup with rustc and cargo (at least 1.96.0)
  ```sh
  rustup upgrade
  ```

### Usage

1. Run the program
   ```sh
   cargo run
   ```
2. Use it like normal shell
